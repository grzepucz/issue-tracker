# issue-tracker
Node.js backend for issue tracker

## installation
``` npm install ```

## dev env
``` npm run watch:dev ```

## prod env
``` npm run prod ```

### Routes

## GET /issue
Listing all issues in db.

## GET /issue/{id}
Get info about issue by id.

## POST /issue
Create new issue.

## PUT /issue/{id}
Update issue by id with parameters passed in request body.

## DELETE /issue/{id}
Delete issue by id.
